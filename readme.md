# Project Aries

CRUD website project. Made using these ingredients:

- Backend
    - SpringBoot
    - WebFlux
    - Spring Security
- Frontend
    - Next.js
    - Mantine UI
- Database
    - mongodb

---

pre-requisites:

- JRE > v21
- mongodb running on localhost:27017
- npm

## How to run backend dev

  ```
    cd ./aries/backend
    ./gradlew bootRun
  ```

## How to run frontend

  ```
    cd ./aries/frontend/ram
    npm run dev
  ```
